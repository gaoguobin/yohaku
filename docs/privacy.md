# Privacy

Goal Shaper is a local Codex plugin and skill package. It does not collect,
store, transmit, or sell personal data on its own.

The plugin shapes prompts inside the user's Codex session. Any data handling by
Codex itself is governed by the user's Codex and OpenAI product settings and
policies.

Goal Shaper does not bundle apps, MCP servers, background services, analytics,
or external credential integrations.
