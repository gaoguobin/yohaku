# Goal Shaper

Goal Shaper turns rough Codex requests into verifiable Goal mode packages.
It helps decide whether Goal mode fits, asks only for missing details that
change success or safety, drafts a copyable `/goal`, and stops.

## Included Skill

- `goal-shaper`: shape fuzzy requests into Codex Goal mode packages with an
  evidence checklist, assumptions, safety boundaries, and durable guidance
  candidates.

## What It Does Not Do

- It does not run `/goal`.
- It does not call Codex runtime goal tools.
- It does not implement code changes.
- It does not write support specs before asking for confirmation.

## Starter Prompts

- Shape my rough task into a safe Codex `/goal`.
- Turn my request into a verifiable Goal mode package.
